================================================================================
                       TROUBLESHOOTING
================================================================================

Issue: "Ollama server not responding"
Solution: Start service with: ollama serve

Issue: "No models found"
Solution: Pull a model with: ollama pull mistral

Issue: Slow inference (>30 seconds)
Solution: Use smaller model (phi) or check system RAM

Issue: Port 11434 already in use
Solution: Ollama is already running (normal - no action needed)

Issue: Health check shows ollamaAvailable=false
Solution:
  1. Verify Ollama is running: ollama serve
  2. Check OLLAMA_BASE_URL in .env
  3. Run test: node test-ollama.js
